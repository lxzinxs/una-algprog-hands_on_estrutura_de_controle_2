# ☕ Desafios de Lógica e Algoritmos em Java

Este repositório contém uma série de exercícios práticos desenvolvidos para o aprimoramento da lógica de programação utilizando a linguagem Java. Os desafios abordam manipulação de saída de dados, geometria esférica, análise de strings e estruturas de repetição controladas.

---

## 📂 Exercícios Desenvolvidos

### 1. 🎨 Arte em Console (`Exercicio_01.java`)
Um programa focado na formatação de saída de texto para criar uma representação visual (ASCII Art) no terminal.
* **Conceito:** Uso de caracteres de escape e concatenação de strings para manter o alinhamento visual.

### 2. 🌍 Cálculo de Distância Geodésica (`Exercicio_02.java`)
Implementação da fórmula para calcular a distância entre dois pontos na superfície da Terra (Haversine simplificada) utilizando coordenadas de latitude e longitude.
* **Fórmula utilizada:** `d = r * acos(sin(x1) * sin(x2) + cos(x1) * cos(x2) * cos(y1 - y2))`.
* **Destaque:** Uso da classe `Math` para conversão de graus em radianos e funções trigonométricas.

### 3. 🔍 Analisador de Strings (`Exercicio_03.java`)
Um algoritmo que recebe uma entrada de texto do usuário e realiza a contagem seletiva de tipos de caracteres.
* **Funcionalidade:** Identifica e quantifica letras e números de forma independente através de um laço `for-each` e métodos da classe `Character`.

### 4. 📝 Quiz Interativo com Controle de Tentativas (`Exercicio_04.java`)
Sistema de múltipla escolha que simula uma questão acadêmica sobre estruturas de controle em Java.
* **Destaque Técnico:**
    * Implementação de validação de entrada com laço `while`.
    * Uso de `switch-case` para processar as alternativas.
    * Lógica para limitar o número de tentativas e informar o desempenho do usuário.
